# Echo 音乐播放器 — 小米米萌百亿 Token 计划申请

## 项目概述

Echo 是一款基于 Electron + React + TypeScript 构建的跨平台桌面音乐播放器，支持 Windows、macOS 和 Linux 系统。项目从零开始自主开发，已具备完整的 v1.0.0 可用版本，代码托管于 GitHub（https://github.com/ubejutat68-cmd/echo-music-player）。

技术架构上，项目采用 Electron 主进程 + React 渲染进程的桌面应用架构，前端以 Vite 5 为构建工具，UI 使用 Material UI 5，状态管理基于 Zustand 4，音频引擎从 Web Audio API 底层自研。全项目使用 TypeScript 进行类型约束，代码结构清晰，模块化程度高。项目还实现了多窗口协同——主窗口负责音乐库管理与播放控制，迷你播放窗口提供精简悬浮操作条，桌面歌词窗口以透明置顶方式悬浮于屏幕底部，三窗口通过 IPC 实时同步播放状态，迷你窗口和歌词窗口可独立于主窗口运行。

核心功能方面，Echo 具备七大功能模块：一是本地音乐管理，支持递归扫描文件夹，自动解析 MP3、FLAC、AAC、WAV、OGG 等主流音频格式的元数据（标题、艺术家、专辑、封面等），采用虚拟滚动技术高效渲染数千首歌曲列表，支持多维度排序与关键词筛选。二是在线音乐聚合搜索，集成网易云音乐、Bilibili、MyFreeMp3 三大音源，用户在统一搜索界面中跨平台检索并直接播放，结果可加入歌单或收藏夹，播放链接自动缓存，在线请求通过主进程代理绕过浏览器 CORS 限制。三是自研音频引擎，基于 AudioContext 实现播放、暂停、切歌、拖拽进度以及四种播放模式（顺序、循环、单曲、随机），内置 10 段参量均衡器（32Hz-16kHz，±12dB 独立可调），提供流行、摇滚、古典、爵士、电子等预设音效及实时频谱可视化。四是歌词系统，支持本地 .lrc 文件解析、音频内嵌歌词读取及网易云在线歌词获取，桌面歌词窗口采用透明无边框设计，绿色荧光文字逐行显示，支持锁定与拖动。五是播放列表与收藏，内置收藏夹，支持自定义歌单的创建、重命名、删除与曲目排序，数据通过 electron-store 持久化存储。六是系统托盘与全局媒体快捷键，支持最小化到托盘及键盘媒体键控制。七是主题系统，支持深色/浅色切换与自定义主题色，配备 Toast 通知与错误边界保护。

与 AI 大模型的融合方面，Echo 具备巨大的 AI 赋能潜力。借助小米米萌大模型，计划实现以下功能：一是智能音乐推荐，基于用户播放历史、收藏偏好和听歌场景，利用大模型进行语义级推荐，实现如"推荐适合下雨天工作时听的音乐"等自然语言描述推荐。二是自然语言搜索，突破关键词匹配局限，支持"像《晴天》那样风格的歌"等模糊语义搜索。三是 AI 歌单生成，用户输入一句话描述即可自动编排歌单。四是歌词翻译与解读，提供高质量的外语歌词翻译和创作背景注释。五是语音控制，实现完全离手操作。六是音乐知识问答，化身随身音乐百科全书。七是个性化音乐报告，自动生成生动的听歌数据报告。

项目计划在获得小米米萌百亿 Token 支持后，分三阶段推进 AI 融合：第一阶段实现自然语言搜索与智能歌单生成，第二阶段上线推荐引擎与歌词翻译，第三阶段集成语音交互与知识问答，最终打造一款真正"懂你"的智能音乐播放器。
