# Echo 音乐播放器 — 小米米萌百亿 Token 计划申请

## 项目介绍

Echo 是一款基于 Electron + React + TypeScript 构建的跨平台桌面音乐播放器，支持 Windows、macOS 和 Linux 三大平台。项目从零开始自主开发，拥有完整的本地音乐管理、在线音乐聚合搜索、桌面歌词、迷你播放器、10 段均衡器、音频可视化等丰富功能，目前已进入可用阶段。

### 技术架构

项目采用 Electron 主进程 + React 渲染进程的经典桌面应用架构，前端使用 Vite 作为构建工具，UI 框架采用 Material UI (MUI) 5，状态管理使用 Zustand，播放引擎基于 Web Audio API 从底层构建。整个项目使用 TypeScript 进行严格的类型约束，代码质量可控。项目实现了多窗口协同架构——主窗口负责音乐库管理和播放控制，迷你播放窗口提供精简的播放条，桌面歌词窗口以透明置顶的方式悬浮在屏幕底部。三个窗口之间通过 IPC（进程间通信）实现播放状态的实时同步，迷你播放器和桌面歌词窗口可独立于主窗口运行。

### 核心功能

**本地音乐管理：** 支持递归扫描文件夹，自动解析 MP3、FLAC、AAC、WAV 等多种音频格式的元数据（标题、艺术家、专辑、封面等），提供按标题/艺术家/专辑/时长/添加日期排序和筛选功能，支持虚拟滚动以高效渲染数千首歌曲的列表。

**在线音乐聚合搜索：** 集成三个在线音乐源——网易云音乐（通过 NeteaseCloudMusicApi）、B 站（通过官方搜索和播放接口提取 DASH 音频流）和 MyFreeMp3（网页解析）。用户可在统一的搜索界面中跨平台检索音乐并直接播放，搜索结果可加入歌单或收藏夹进行持久化管理。播放链接自动缓存，避免重复请求。整个在线请求通过主进程代理，绕过浏览器的 CORS 限制。

**自研音频引擎：** 基于 Web Audio API 的 AudioContext 构建，支持本地文件解码播放和在线音频流式加载，实现了播放/暂停/切歌/拖拽进度等基础功能，以及四种播放模式（顺序、循环、单曲循环、随机）。内置 10 段参量均衡器（32Hz-16kHz），提供流行、摇滚、古典、爵士、电子等预设音效，每段独立可调（±12dB）。同时具备基于 AnalyserNode 的实时频谱可视化。

**歌词系统：** 支持解析本地 .lrc 歌词文件（含嵌入式时间标签和元信息过滤）、读取音频文件内嵌歌词、以及在线获取网易云歌词。桌面歌词窗口采用透明无边框设计，以绿色荧光文字逐行显示当前和下一句歌词，始终置顶，支持锁定/解锁拖动。

**播放列表与收藏：** 内置"我最喜欢"收藏夹，支持创建、重命名、删除自定义歌单，歌单内可重新排序、移除曲目。所有歌单和偏好设置通过 electron-store 持久化到磁盘，启动时自动恢复。

**主题与体验：** 支持深色/浅色主题切换和自定义主题色，系统托盘常驻（支持最小化到托盘），全局媒体快捷键（播放/暂停、上一首、下一首），右键上下文菜单，Toast 通知系统，错误边界保护等。

### 与 AI 大模型的结合前景

Echo 音乐播放器作为一个成熟的桌面应用平台，具备与 AI 大模型深度融合的巨大潜力。以下是我们计划借助小米米萌大模型实现的功能方向：

**1. 智能音乐推荐与发现：** 基于用户当前的播放历史、收藏偏好和听歌时段，利用大模型进行语义级别的音乐推荐。不同于传统的协同过滤算法，大模型可以理解音乐的"情绪"和"氛围"维度，实现"想听一首适合下雨天工作时的背景音乐"这样的自然语言推荐。

**2. 自然语言交互搜索：** 当前的音乐搜索局限在歌曲名和艺术家关键词匹配。接入大模型后，用户可以用自然语言描述想听的音乐，如"播放一首像《晴天》那样风格的歌"或"推荐 2000 年代华语摇滚经典"，模型理解用户意图并精确匹配。

**3. 智能歌单生成：** 用户只需输入一句话描述，如"为我创建一个适合夜跑时听的 30 分钟歌单"或"帮我整理一个周末做家务时的轻松播放列表"，大模型即可根据音乐的风格、节奏、情绪等特征自动编排歌单。

**4. 歌词理解与翻译：** 利用大模型的翻译和文本理解能力，为外语歌曲提供高质量的中文翻译歌词，并可生成歌词的逐句解读、背景故事和文化注释，提升用户的音乐欣赏体验。

**5. 语音控制：** 结合语音识别，实现完全的语音操控——"下一首"、"音量调到 50%"、"播放周杰伦的歌曲"、"我喜欢这首歌，加入收藏"，让用户在驾驶或做家务等场景中解放双手。

**6. 音乐知识问答：** 在播放歌曲时，用户可以随时向 AI 提问——"这首歌的创作背景是什么？"、"这位歌手还唱过哪些类似的歌？"、"这首歌使用了哪些乐器？"——形成一个随身的音乐百科全书。

**7. 个性化音乐日报/周报：** 大模型根据用户的听歌数据自动生成有趣的音乐收听报告，包含听歌时长统计、最爱的风格/艺人/时段分析、音乐情绪曲线等，用生动的语言呈现。

### 项目现状与规划

Echo 当前已完成核心功能的开发和调试，具备完整可用的 v1.0.0 版本。项目计划在获得小米米萌百亿 Token 支持后，优先实现智能搜索与自然语言推荐功能，并逐步扩展语音交互和 AI 歌单生成等高级特性，打造一款真正"懂你"的智能音乐播放器。

我们相信，Echo + 小米米萌 = 下一代智能音乐体验。
